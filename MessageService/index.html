MessageService

REST API for Timestamped Greetings

Service is running

Welcome to the MessageService API! This is a simple REST API built with ASP.NET Web API 2 that provides timestamped greeting messages. The service is currently running and ready to accept requests.

Features

  • RESTful API endpoint for getting timestamped messages
  • Swagger/OpenAPI documentation for easy testing
  • CORS enabled for cross-origin requests
  • JSON response format

API Endpoint

GET /api/message

Returns a "Hello World" message with the current date and time prepended.
